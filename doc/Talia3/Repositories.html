Overview

Represents a set of repositories.

Repositories are stored and accessed through the name (symbol) associated with them in the Talia3 configuration.

Repositories will always include the name :local, which is Talia3 default repository.

Should mostly be used through methods of HasRepositories mixin.

def [](name)
  raise Exception, "Repository #{name} not declared for this object." unless include? name
  raise Exception, "Repository :#{name} is not defined in talia3.yml for env #{Rails.env}." unless self.defined? name
  Talia3.repository name
end
