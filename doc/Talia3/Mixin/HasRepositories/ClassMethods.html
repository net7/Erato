- (Symbol) default_repository=(name)

Sets a default repository to use for all objects of this class.

Also adds the repository name to the class’ repositories list.

*Note*: if the name does not belong to a valid repository, no exceptions are raised until the repository is actually used by an object with #use_repository or #get_repository.

def repository(name=nil)
  raise Exception, "No default repository for #{name}" unless name or default_repository
  repositories[(name.nil? ? default_repository : name)]
end
