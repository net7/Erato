Overview

Mixin for controllers, implementing user authentication.

IMPORTANT: All these methods are private to the controller, if you want to use them in the view, You will need to either use #controller.send there, or use the methods in the controller and store the result in attributes.

Examples:

Including this mixin: 

class Mycontroller
  include Talia3::Mixin::AuthController
  ...
end

Log a user in: 

log_in @user

Checking if the user is currently logged in: 

logged_in?

Retrieving a user from session: 

logged_user

create an authentication filter: 

before_filter :authenticate

create an authentication filter for administrators only: 

before_filter :except => [:init] do |controller|
  controller.send :authenticate, true
end
