Overview

Authentication token for Talia3

Use the Talia3::Auth module for a better interface to this object.

Examples:

Get a token’s hash: 

Talia::Auth::Token.token
Talia::Auth::Token.to_s

Create and save a token for a user: 

Talia3::Auth::Token.create!(:user => @user, :ip => '192.0.32.10')

Retrieve a token’s user: 

Talia::Auth::Token.user '192.0.32.10', "e4b6505b5d0451d847371d50536f2f9c6cf1416e383f84041d7b1a0c0f52c3f0"

Prolong a token’s life: 

Talia::Auth::Token.touch '192.0.32.10', "e4b6505b5d0451d847371d50536f2f9c6cf1416e383f84041d7b1a0c0f52c3f0"

def self.user(ip, token)
  result = fetch ip, token.to_s
  return nil if result.nil?
  # Removes the token if it is too old. As we are already there,
  # _all_ old tokens are removed.
  clear_old and return nil if result.updated_at < end_time
  (result.nil? or result.user.nil?) ? nil : result.user
end
