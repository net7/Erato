Overview

Special namespace to be used as contexts.

Uses Talia3 configuration to determine the base uri. Defaults to “#Talia3::N::LOCAL.to_scontexts/“if no configuration is found.

Examples:

Talia::N::CONTEXT.ontology

def initialize(prefix=nil, uri_s=nil)
  @contexts = Talia3.config(:contexts) || {}
  @uri = Talia3::Namespace::LOCAL.new.uri_for("contexts/").to_s
  @prefix = :contexts
end
